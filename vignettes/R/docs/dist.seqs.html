dist.seqs {Metagenomics} R Documentation

create sequence distance matrix

Description

Usage

dist.seqs(align.fasta,
    calc = onegap,
    countends = F,
    cutoff = 0.03,
    output = lt,
    num.threads = 32,
    logfile = [10]dist.seqs.txt);

Arguments

Details

The dist.seqs command will calculate uncorrected pairwise distances between aligned DNA sequences. This approach is better than the commonly used DNADIST because the distances are not stored in RAM, rather they are printed directly to a file. Furthermore, it is possible to ignore “large” distances that one might not be interested in. The command will generate a column-formatted distance matrix that is compatible with the column option in the various cluster commands. The command is also able to generate a phylip-formatted distance matrix.
