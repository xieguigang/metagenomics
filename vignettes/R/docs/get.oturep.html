get.oturep {Metagenomics} R Documentation

Get the representative sequence

Description

Usage

get.oturep(dist, contig.unique.fasta, list,
    label = 0.03,
    logfile = [13]get.oturep.txt);

Arguments

Details

While the bin.seqs command reports the OTU number for all sequences, the get.oturep command generates a fasta-formatted sequence file containing only a representative sequence for each OTU. A .rep.fasta and .rep.name file or .rep.count_table file is generated for each OTU definition.
