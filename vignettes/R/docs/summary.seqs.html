summary.seqs {Metagenomics} R Documentation

The summary.seqs command will summarize the quality of sequences in an unaligned or aligned fasta-formatted sequence file.

Description

Usage

summary.seqs(seqfile, count.table,
    num.threads = 8,
    logfile = log.txt);

Arguments

num.threads

The processors option enables you to accelerate the summary process by using multiple processors. Default processors=Autodetect number of available processors and use all available.
