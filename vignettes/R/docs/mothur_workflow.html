mothur_workflow {Metagenomics} R Documentation

commandline workflow for run mothur

Description

Usage

mothur_workflow(work16s);

Arguments

work16s

the 16s workflow workspace object, contains necessary parameter files.

Details

you must change of current work directory to the output directory, due to the reason of all of the data file path in this workflow function is relative path to the result data directory outputdir.

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

This function has no value returns.
