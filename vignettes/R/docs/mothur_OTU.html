mothur_OTU {Metagenomics} R Documentation

Build OTU based on mothur

Description

Usage

mothur_OTU(left, right, refalign,
    outputdir = ./,
    assembler = list(algorithm = needleman, score = [1, -1, -2, -1], insert = 20),
    num.threads = 32);

Arguments

left

the left paired-end sequence file.

right

the right paired-end sequence file.

outputdir

the result output dir for save temp file

num.threads

the number of the processor threads that will be used in the mothur software.
