mothur_files {Metagenomics} R Documentation

Create mothur file list

Description

Usage

mothur_files(dir,
    filename = ./16s.files);

Arguments

dir

a directory path which contains the R1/R2 pair-end fastq sequence file.

filename

the file name to save as the 16s file list.

Details

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

a character vector of the sample id in the given data directory.
